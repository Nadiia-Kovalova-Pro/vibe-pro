# Directory Structure

## Main Directories

### `/backend`
Contains the Flask REST API application, including:
- SQLite database configuration
- API routes and controllers
- Database models and migrations
- Authentication with JWT
- Swagger API documentation
- Python dependencies and virtual environment setup

### `/frontend`
Houses the React TypeScript application, featuring:
- React components and routes
- TypeScript configurations
- TailwindCSS styling
- API integration with Axios
- State management (Zustand, React Query)
- UI components with ShadcnUI
- Node.js dependencies

### `/preview`
Contains application screenshots and preview images showing:
- REST API documentation views
- Frontend UI screenshots 
- Application workflow demonstrations
- Interface mockups
